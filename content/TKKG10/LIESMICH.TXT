1. Allgemeine Hinweise
==================

Damit du das Spiel "TKKG - Panik im Internat" optimal spielen
kannst, muss dein Monitor auf 16bit (PC) bzw. 32768 (MAC) Farben eingestellt sein.

Vor dem Start der CD-ROM solltest du alle anderen Programme schließen. Dann läuft das
Programm schneller.

Die CD-ROM sollte sich während des gesamten Spiels im CD-ROM-Laufwerk befinden.

Bitte beachte die Mindestanforderungen für deinen Computer, um dieses Spiel spielen
zu können:
Windows: 	* Win 95 / 98 / NT 4.0 / 2000 / ME / XP
		* Pentium 166 MHz
		* SVGA-Grafikkarte (16 Bit)
		* Soundkarte
		* CD-ROM-Laufwerk (4-fache Geschwindigkeit)
		* mind. 32 MB RAM verfügbar
		* mindestens 40 MB freier Festplattenspeicher (kleine Installation)
		

Macintosh:	* Mac OS 8.1 bis Mac OS 9.1
		* Power PC
		* Grafikkarte (32768 Farben)
		* Soundkarte
		* CD-ROM-Laufwerk (4-fache Geschwindigkeit)
		* mind. 32 MB RAM verfügbar
		* mindestens 40 MB freier Festplattenspeicher



=====================
2. Installation der CD-ROM
=====================

Bei der Installation solltest du folgendes beachten:

Windows:
Das Installations-Programm startet automatisch, wenn du "TKKG - Panik im Internat" in das
CD-ROM-Laufwerk legst.

Falls die Funktion "Autostart" in Windows ausgeschaltet ist, kannst du die CD-ROM so
starten:
Gib im Menü "Datei/Ausführen" den Befehl "X:setup.exe" ein. Setze für "X" den
Laufwerksbuchstaben deines CD-ROM-Laufwerks ein. Klicke auf "OK" und folge den Anweisungen.

Übrigens: Du kannst jederzeit mit dem De-Installationsprogramm alle installierten Daten
von deiner Festplatte löschen. Dazu musst du im Startmenü/Programme/Tivola/TKKG10 
das Icon "TKKG10 - Entfernen" klicken und den Anweisungen folgen.


Macintosh:
Nachdem du "TKKG - Panik im Internat" in das CD-ROM-Laufwerk gelegt hast, öffnet sich
automatisch ein Fenster. Kopiere den Ordner "TKKG 10" auf deine Festplatte. 
Wenn du in diesem Ordner das Icon "TKKG 10" anklickst, wird das Spiel gestartet.

Um das Programm wieder von der Festplatte zu löschen, ziehst du die Datei "TKKG 10" 
von der Festplatte auf den Papierkorb.
